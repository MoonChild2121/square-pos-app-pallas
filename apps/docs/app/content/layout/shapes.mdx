# Shapes
<Paragraph size={'extraLarge'} color={'tertiary'}>Simple geometric shape components for creating circular and square UI elements.</Paragraph>
<ComponentPreview name="shapes" />
</Section>

<Section>
## Usage

```jsx
import { Circle, Square } from '@styled-system/jsx'
```

```jsx
<Circle className={css({ size: '100px', bg: 'fill.accent' })}>
  Circle Content
</Circle>

<Square className={css({ size: '100px', bg: 'fill.secondary' })}>
  Square Content
</Square>
```
</Section>

<Section>
## Properties

Both Circle and Square components accept the following properties:

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `size` | `string \| number` | - | Sets both width and height to the same value |

Additionally, they accept all common layout properties through the className prop with css function.

### With Content

Both Circle and Square can contain any content:

```jsx
<Circle className={css({ size: '150px', bg: 'fill.accent', color: 'white' })}>
  <Flex className={css({ direction: 'column', align: 'center' })}>
    <Box as="span" className={css({ fontSize: 'xl', fontWeight: 'bold' })}>75%</Box>
    <Box as="span" className={css({ fontSize: 'sm' })}>Complete</Box>
  </Flex>
</Circle>
```

### As Avatars

Circles work well for avatar displays:

```jsx
<Circle className={css({ 
  size: '60px', 
  overflow: 'hidden', 
  border: '2px solid', 
  borderColor: 'accent.500' 
})}>
  <img 
    src="/avatar.jpg" 
    alt="User Avatar"
    className={css({ 
      width: '100%', 
      height: '100%', 
      objectFit: 'cover' 
    })}
  />
</Circle>
```
