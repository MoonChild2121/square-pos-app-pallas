# Getting Started

This guide will help you set up Pallas UI in your React project.
</Section>

<Section>
## Installation

Install Pallas UI using your package manager of choice:

```bash
# Using npm
npm install @pallas-ui/react

# Using yarn
yarn add @pallas-ui/react

# Using pnpm
pnpm add @pallas-ui/react
```
